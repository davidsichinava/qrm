*
* This file includes replication details for the analyses presented in:
*
* Iqbal, Zaryab, and Christopher Zorn.  2008.  "The Political 
* Consequences of Assassination."
* 
* This replication archive includes four files:
* 
* IqbalZornJCR.dta -- the replication data in Stata 7/8/9 format
* IqbalZornJCR.txt -- the replication data in plain-text (space-
*                  -- delimited) format
* IqbalZornJCR.log -- the replication log
* readme.txt       -- this file
*
* Variables in the data are:
*
* ccode                          COW country code* year                           Year* UNREST                         Banks-based first factor score* INSTAB                         Banks-based second factor score* internalconflict               Internal Conflict (TYPE3)* internationalizedconflict      Internationalized Conflict (TYPE4)* combinedconflict               internalconflict + internationalizedconflict* logpop                         ln(POPUL)* YEAR                           Year - 1900* coldwar                        1 if cold war year, 0 otherwise* lngdplag                       Lagged LNGDP1000* lngrowlag                      Lagged LNGROWTH* lnopenlag                      LNOPEN lagged* POLITYLAG                      POLITY01 at t-1* POLITYLAGSQ                    POLITYLAG squared* regulated                      1 if succession is regulated, 0 else* assass                         Assassination indicator* asslag                         ASSASS lagged one year* asslagxreg                     ASSLAG * REGULATED* natdeath                       1 if leader died a natural death* natdeathlag                    Lagged NATDEATH* natdeathlagxreg                NATDEATHLAG * REGULATED
